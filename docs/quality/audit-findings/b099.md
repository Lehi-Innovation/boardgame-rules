The actual 4th loss condition — Abandoned — is a meaningful win-coordination rule that would directly affect player decisions at the table.

### MAJOR FINDING 2 — Missing Game-End Triggers

Summary: "The game ends at the end of the round when any player reaches Fort Level 5 (all players get equal turns)."
Source: "The end of the game triggers in any one of **these three ways**: Any player has 25 or more victory points on the Victory Track / Any player reaches Fort Level 5 / The Park deck is empty (even if the Park still has cards)"

CONTRADICTION: Summary lists only 1 of 3 game-end triggers. The 25+ VP trigger and empty Park deck trigger are completely missing. Both are meaningful — a player could win without ever reaching Fort Level 5, or the game could end via deck exhaustion.

### MAJOR FINDING 3 — Invented Tiebreaker

Summary: "Tiebreaker: highest Fort Level, then most resources in Pack."
Source: "If there is a tie, the tied player with the highest Fort Level wins. If still tied, share the victory."

CONTRADICTION: The summary invents "most resources in Pack" as a second tiebreaker. The source says tied Fort Levels result in shared victory, not a Pack resource comparison. This would cause players to determine incorrect winners in tied games.
