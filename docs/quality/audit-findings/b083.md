## earth-reborn

**Verdict: PASS**

Source: Esoteric Order of Gamers rules summary PDF (third-party), confirmed to cover Earth Reborn by Z-Man Games (2010).

Prefilter returned OK. Source is a comprehensive third-party rules summary.

### Verification

**Victory/game-end:** Summary states "Victory is determined by scenario-specific objectives." Source: "Check for victory conditions by referring to the scenario rules." Matches — the game is scenario-driven and no fixed end condition exists outside scenarios.

**Turn structure:** Summary states 3 phases: Initiative, Activation, Final. Source confirms same 3 phases in same order.

**Order colors:** Summary: "Black orders = activated characters only. Red and Gold orders = activated or interrupting characters. Gold orders can also be used in combined orders." Source (line 510-511): "Black Orders can only be used for activated characters, while red and gold Orders can be used for activated characters and interrupting characters." Source (line 961): "other characters can use gold order sections so they can all collectively perform a combined order." Matches.

**Activation options:** Summary lists: Activate Character, Reactivate, Draw Tile (1 CP), Choose from Discard (2 CP), Pass. Source confirms all five options with same CP costs. Matches.

**Interruption:** Summary: "Win duel -> Assign 1 Order tile + execute up to 2 red/gold orders." Source (line 503-504): "Execute up to 2 red or gold colored Interrupting orders from any Order tiles assigned to the interrupting character." Matches.

**Capture:** Summary: "Execute Close Combat against an adjacent wounded enemy. If damage would kill, the character is captured instead." Source (line 872-880): "execute a normal Close Combat order against an adjacent, wounded enemy... If the damage rolled would have been sufficient to kill the victim, he is captured." Matches. Summary: "Prisoner placed on captor's card, counts as weight 1." Source (line 882-884): "place the victim's figure on the captor's character card... A prisoner has a weight value of 1." Matches.

**Torture:** Summary: "Salemites can torture prisoners for Spy Points." Source (line 947): "Only Salemites can torture prisoners to obtain Spy Points (SP)." Matches.

**Combination orders:** Summary: "Activated character uses any color; others must use gold sections." Source (line 969-972): "The order section he executes can be of any color, but the color of the order sections executed by all other combined characters must be gold." Matches.

**Radio Scrambling:** Summary: "Switch ON a radio scrambler to place scrambling markers on enemy characters, potentially limiting their CP usage." Source confirms radio scrambling limits CP based on CP Bonus value. Matches.

**Close combat - defender wounds:** Summary: "Defender can inflict wounds on attacker only via critical hits." Source (line 282): "The defender can inflict wounds on the attacker with critical hits, but not with normal hits." Matches.

**Character death:** Summary: "Healthy LP exhausted -> flip to Wounded -> Wounded LP exhausted -> dead." Source (lines 299-308) confirms this sequence. Matches.

**Final Phase order tiles:** Summary: "Discard Order tiles in excess of 5 from behind screens." Source (line 194-195): "All players discard Order tiles in excess of 5 from behind their screens, and may discard all or part of their remaining tiles." Matches.

**Numbers checked:** CP costs for Draw (1), Choose from Discard (2), Interruption orders (up to 2), Torture dice per CP (1), weight of prisoner (1), close combat arc colors (dark-blue/light-blue/grey).

## eclipse

**Verdict: MAJOR**

Source: Eclipse rulebook (confirmed — matches all component counts and designer details).

### Prefilter
Prefilter returned OK.

### Finding 1: Technology category bonus — thresholds wrong by 2 (MAJOR)

**Summary (line 88):** "Technology category bonus: 1/2/3/5 VP for having 2/3/4/5+ technologies in a single category"

**Source (lines 1547–1549):** "Progress on the Technology Tracks: 4 Technology Tiles on a track = 1 VP, 5 tiles = 2 VP, 6 tiles = 3 VP, 7 tiles = 5 VP"

The summary says bonuses start at 2 tiles. The source says bonuses start at 4 tiles. Every threshold is off by 2. Players using the summary would think they earn VP from just 2 technologies in a category when the actual threshold is 4.

### Other claims verified

**Game end:** Summary: "After 9 rounds, score Victory Points... Highest VP total wins." Source (line 1539-1540): "The game ends after the 9th round. The player with the most Victory Points is the winner." Matches.

**Tiebreaker:** Summary: "Tiebreaker: most Resources remaining." Source (lines 1552-1554): "In case of a tie, the total amount of Resources (Money, Science and Materials) in each tied player's Storage is the tie breaker." Matches.

**Scoring sources:** Summary lists: Controlled hexes, Discovery Tiles (2 VP each), Reputation Tiles, Ambassador Tiles (1 VP each), Monoliths (3 VP each), Technology category bonus, Traitor Card (-2 VP). Source (lines 1541-1550) lists same sources. Monolith = 3 VP: source line 1546: "Monoliths on own hexes (3 VP per Monolith)." Matches. Ambassador = 1 VP: source line 699: "Ambassador Tiles on your Reputation Track are worth 1 Victory Point." Matches. Discovery Tile = 2 VP: source line 390: "the back side has a 2 Victory Point value." Matches. Traitor = -2: source line 1119: "-2 Victory Point penalty." Matches.

**Turn structure (4 phases):** Summary: "Action Phase, Combat Phase, Upkeep Phase, Cleanup Phase." Source (lines 503-513) confirms same 4 phases. Matches.

**First-to-pass rule:** Summary: "First to pass gets the Starting Player Marker for next round." Source (line 469): "The first player to pass gets the Starting Player Marker for the next round." Matches.

**Build action:** Summary: "Build up to 2 Ships or Structures." Source (line 875): "Build up to two Ships (Interceptors, Cruisers, Dreadnoughts and Starbases) or Structures (Orbitals and Monoliths)." Matches.

**Upgrade action:** Summary: "return any number of Ship Parts to supply, then take up to 2 new Ship Parts." Source (lines 855-856): "return as many Ship Part Tiles to the Supply Board as you want. Then take up to two tiles." Matches.

**Move action:** Summary: "Make up to 3 ship movements." Source (lines 941-942): "you may do up to three Ship movements." Matches.

**Influence action:** Summary: "Move up to 2 Influence Discs... Also flip 2 used Colony Ships face up." Source (line 623): "you may move up to two Influence Discs." Source (line 639): "you may turn two of your used Colony Ships face up." Matches.

**Dreadnought cost:** Summary: "Dreadnought (8)." Source (line 892): "Dreadnought 8." Matches.

**Diplomacy:** Summary: "4+ players only... Breaking diplomacy (attacking) gives you the Traitor Card (-2 VP)." Source confirms diplomacy and Traitor Card penalty. Matches.

## egizia

**Verdict: MAJOR**

Source: Egizia rulebook (Italian title "Acchittocca — In the valley of the kings"). OCR quality is poor in the first 500 lines but improves in the rules section. Source confirmed authentic: same designers (Virginio Gigli, Flaminia Brasini, Stefano Luperto, Antonio Tinto), 2-4 players, 90 minutes.

### Prefilter
Prefilter returned WARN SOURCE_SUSPECT (title tokens absent from first 3000 chars). After reading first 60 lines, confirmed: Italian title on line 1 ("Acchittocca / In the valley of the kings"), designers listed on lines 6-7 match summary frontmatter exactly. Source is authentic.

### Finding 1: Joker construction crew rule — completely reversed (MAJOR)

**Summary (line 138):** "The Joker construction crew can only be used alone, never with another crew."

**Source (line 614):** "The player can never use the Joker construction crew alone. He can use it once per round with another construction crew and then must, likewise, turn it over."

This is a direct reversal. The summary says the Joker can only be used alone. The source says it can *never* be used alone and must always be combined with another crew. This completely inverts the Joker's role. Players following the summary would play the Joker as a solo crew and never combine it, which is the opposite of the actual rule.

### Finding 2: Stone market conversion rate in Final Scoring section — contradicts source and own Special Rules section (MAJOR)

**Summary Final Scoring (line 126):** "If a player's marker is on one of the bottom two spaces, they may sell remaining stones for 1 point per stone (up to 10, rounded down)."

**Source (lines 755–756):** "If the player's marker is on one of the bottom two spaces of the stone market, he may sell his remaining stones. For each 2 stones he sells, he scores 1 point (rounded down)."

**Summary Special Rules (line 141):** "At game end, any player with their marker on the bottom or second-to-bottom space of the stone market may convert remaining stones to points at a rate of 2 stones for 1 point."

The Final Scoring section of the summary says "1 point per stone" — double the actual rate. The source and the summary's own Special Rules section both say "2 stones for 1 point." Players reading the Final Scoring section would score twice as many points from stone sales. Note: "up to 10, rounded down" appears to be blended from a Nile card ability (source line 915: "The player can sell stones for 1 point per stone up to 10"), which is a different, special-card-only rule.

### Other claims verified

**Cooperation bonus table:** Summary: "1 ship: 1 point, 2 ships: 3 points, 3 ships: 6 points." Source (lines 703-708): "1 ship 1 point, 2 ships 3 points, 3 ships 6 points." Matches.

**Win condition:** Summary: "The player with the most points wins. Tiebreaker: the player whose stone is in front on the scoring track." Source (lines 770-771): "The player with the most points is the winner. If players are tied, the player whose stone is in front on that space of the scoring track is the winner." Matches.

**Rounds:** Summary: "5 rounds." Source (line 503): "In each round, the players pass through 7 phases, then the next round begins" (implying multiple rounds). The 5-round count is stated in the summary's Player Reference and Setup sections. Source confirms 5 rounds via card distribution rules (rounds 1-2, 3-4, 5 use different card sets). Matches.

**Phase order:** Summary: "Lay out Nile cards > Place ships > Feed crews > Produce stones > Build > Co-operation bonus > Determine player sequence." Source (lines 506-512) lists same 7 phases in same order. Matches.

**Ship placement rule:** Summary: "A player must always place their new ship further down-river (toward the delta) from their already-placed ships." Source (line 529): "a player must always place his new ship further down-river from his already-placed ships." Matches.

**Scarab bonus:** Summary: "For each 10 points already scored (on the scoring table), gain an additional point." Source (lines 983-984): "he scores an additional point for each 10 points he has on the scoring table (rounded down)." Matches.

**Grave tile scoring triggers:** Summary: "1-10 = 2 points, 11-20 = 5 points, 21+ = 9 points." Source (lines 763-767) shows the range labels (1-10, 11-20, 21 or more) but the actual point values were lost to OCR — table columns not preserved. Cannot verify or contradict these numbers from the available source text.

**Final Scoring stone market eligibility:** Summary and source both say "bottom or second-to-bottom space." Matches on eligibility condition.

**2-3 player rule:** Summary: "With 2 or 3 players, use only 2 spaces at each building site." Source (line 774): "With 2 or 3 players only use 2 spaces at each building site." Matches.

**Nile card ordering by round:** Summary: "In rounds 1-2, draw from the 1/2 supply; in rounds 3-4, from the 3/4 supply; in round 5, from the 5 supply." Source (line 187-199, setup instructions) separates three card sets (1/2, 3/4, 5). Matches.
