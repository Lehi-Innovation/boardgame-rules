### Source Notes

The extracted text is a heavily OCR-scrambled scan. Card text and values appear mixed in with gameplay description text. This makes verification of the card table particularly difficult. However, the errors found are clear enough.

### Finding 1 — Juliet's ability is incorrectly described (MAJOR)

Summary: "Juliet | 14 | Alone: value 5; together with another card value 10+: value 15"

Source analysis: The text "Alone: 5, together with Juliet: 15" appears twice in the source — first for Kardinal (lines 6–9): "Kardinal Kardinal Alone: 5, together with Juliet: 15" and again for Alchemist (lines 67–70): "Alchemist Alchemist Alone: 5, together with Juliet: 15."

Juliet's source entry (lines 37–39): "Juliet Juliet 14 14" — only value 14 is stated. The surrounding cut-off text reads "with value 10 or higher. Two or more" which appears to describe Juliet's triggering condition for OTHER cards that get bonuses when together with Juliet, not an ability belonging to Juliet herself.

Interpretation: The source says that Kardinal and Alchemist each have value 5 alone but value 15 when together with Juliet. Juliet herself has a fixed value of 14. The summary inverts this: it claims Juliet has value 5 alone and 15 together with a 10+ card — which is the ability of Kardinal/Alchemist, not Juliet. This means the summary's card table scrambles Juliet's ability. A player using this summary would misplay Juliet (treating her as context-dependent rather than fixed at 14) and Kardinal/Alchemist (not knowing they gain value from pairing with Juliet).

Verdict: MAJOR — misattributed card ability would directly mislead how Juliet, Kardinal, and Alchemist are played.

### Finding 2 — "Gold card" mentioned in scoring but absent from components (MINOR)

Summary: "The Crown card and Gold card each award 3 bonus points to their holder."
Source (line 185–186): "The owners of the Crown and Gold cards each score 3 additional points."

Components section of summary lists only "1 Crown Card" — no Gold card. The source components (lines 12): "• 1 Crown Card" — also no Gold card. Neither the source nor the summary explains what the "Gold card" is. This appears to be a game element not covered in this single-sheet rulebook (possibly won during play). Not an error in the summary per se — the summary accurately reflects the source text — but the incompleteness could mislead. This is MINOR since it copies the source literally.
