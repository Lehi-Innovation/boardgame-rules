# Audit Findings — Batch b088

Games: exit-the-game-the-abandoned-cabin, exit-the-game-the-forbidden-castle,
exit-the-game-the-forgotten-island, exit-the-game-the-mysterious-museum,
exit-the-game-the-pharaoh-s-tomb, exit-the-game-the-polar-station,
exit-the-game-the-secret-lab

All games in this batch are entries in the EXIT: The Game series by Inka & Markus Brand. The
rulebooks are nearly identical in structure: cooperative real-time escape-room puzzle with a decoder
disk, riddle/answer/help card system, and a scoring table based on time + help cards used. All
sources confirmed as correct rulebooks. Most procedural rules are shared across entries; the scoring
table is where individual games diverge.

## 4. exit-the-game-the-mysterious-museum

**Verdict: MAJOR**

### Finding 1 — Scoring table completely wrong (all values fabricated)

**Summary scoring table:**
```
         0 help    1-2    3-5    6-10   >10
< 60:    10         8      —      —      —
< 90:     9         6      3      —      —
<120:     8         6      3      2      —
>120:     —         2      —      1      —
```

**Source scoring table (lines 198–216):**
```
         No       1-2    3-5    6-10   >10
< 60:    10        9      —      —      —
< 90:     9        4      —      —      —
<120:     4        3      —      —      —
>120:     3        2      —      —      —
```

This table is entirely different from the standard EXIT scoring table used by other entries in the series. The summary copied the standard table instead of the Mysterious Museum's actual, more compressed table. Every value the summary gives except "10 stars / no help / <60 min" and "9 stars / no help / <90 min" and "2 stars / 1-2 / >120 min" is wrong or invented.

Specific contradictions:
- <60 min, 1-2 help: summary=8 stars; source=9 stars
- <90 min, 1-2 help: summary=6 stars; source=4 stars
- <90 min, 3-5 help: summary=3 stars; source=blank (only 2 columns have values)
- <120 min, 0 help: summary=8 stars; source=4 stars
- <120 min, 1-2 help: summary=6 stars; source=3 stars
- <120 min, 3-5 help: summary=3 stars; source=blank (invented)
- <120 min, 6-10 help: summary=2 stars; source=blank (invented)
- >120 min, 0 help: summary=— (blank); source=3 stars (missing)
- >120 min, 6-10 help: summary=1 star; source=blank (invented)

A group scoring their game using this summary would compute incorrect star totals in almost every scenario.

### Finding 2 — Setup: strange items placement contradicted

**Summary:** "Leave the two strange items in the box."

**Source (lines 55–56):** "Pull the strange 'tot' item and the string out of the box and place them to the side. You will need these materials later in the game."

The source explicitly instructs players to remove the strange items from the box and place them on the table at setup. The summary says the opposite. Players following the summary could fail to have a required physical object available when the game directs them to use it.

### Finding 3 — Mandatory sequential solving order not documented

**Summary** describes play as collaborative exploration without sequential constraints; Special Rules does not mention ordering.

**Source (lines 165–168):** "IMPORTANT: You have to solve the riddles in the proper order! In other words, you are only permitted to continue with the next riddle or turn to the next page of the book if you have solved the previous code and the instructions tell you that you may do so!"

This is a fundamental structural difference from other EXIT games, explicitly marked IMPORTANT. The Mysterious Museum has mandatory sequential solving; players cannot freely jump between riddles as in other entries. The summary's "Turn Structure" step "Progress or Retry — Follow instructions or keep working" does not capture this constraint.

NUMBERS_CHECKED: 14, SUPPORTED: 2 (scoring section nearly fully wrong)

< 90:     9        7      6      3      —
