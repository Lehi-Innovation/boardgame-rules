## fighting-formations-grossdeutschland-motorized-infantry-division

**Verdict: MAJOR**

### Finding 1 — Invented "40 initiative points pool" mechanic

**Summary:** "A pool of **40 initiative** points is shared between players."

**Source:** No such pool exists. The source describes an Initiative Track with a pawn that moves back and forth. The track has a "0" space in the center and extends to "20" on each side. The pawn's position indicates which player has the Initiative. Source (line 305-309): "The Initiative pawn is placed on the Initiative Track at the start of every scenario, and will move back and forth along the track – that is, towards and away from each player – as Initiative points are spent during play."

No support found for "a pool of 40 initiative points shared between players" (searched: "pool", "40 initiative", "shared", "initiative points").

This is a MAJOR misrepresentation of the core mechanic.

### Finding 2 — Wrong turn-end condition

**Summary:** "When both players pass or the initiative pool is balanced, the turn ends."

**Source (line 880-882):** "A turn ends – and the Turn marker is advanced one space forward along the Time Track – at the end of any order in which there are exactly zero cubes remaining on the Order Matrix."

The turn ends when ALL TEN order cubes are exhausted from the Order Matrix, not when players pass or initiative is balanced. Players can pass on a specific order without ending the turn (source line 863-865: "A player may 'pass' and do nothing instead of performing a chosen order—though the initiative cost for removal of the order cube is still paid.").

### Finding 3 — Wrong/invented order list; missing Advance and Sniper orders; invented "Command" order

**Summary lists orders as:** Move, Fire, Assault, Rally, Command, Asset

**Source order types (table of contents, rules 31–38):** Advance, Assault, Asset, Fire, Move, Rally, Sniper, Support

Specific errors:
- **Advance** order (rule 31) is absent from the summary entirely.
- **Sniper** order (rule 37) is absent from the summary entirely.
- **Support** order (rule 38 — draw asset cards from deck) is renamed "Command" in the summary.
- "**Command**" is listed as an order "Activate HQ abilities, call for support" — no such order exists. "Command" in the source is a mechanic (Command markers placed on the map for command radius benefits), not a named order.

Source (table of contents, lines 80-91): "31. Advance … 32. Assault … 33. Asset … 34. Fire … 35. Move … 36. Rally … 37. Sniper … 38. Support"

Source (line 2010-2011): "Each Support order allows the active player to draw a certain number of assets from the top of his deck (as indicated on his Order Matrix)."

### Finding 4 — Initiative flow description is wrong

**Summary:** "each order type costs a different amount of initiative (taken from the acting player's pool and given to the opponent)"

**Source (lines 1435-1437):** "1) Remove any one order cube from the Order Matrix. 2) Move the Initiative marker a number of spaces towards his opponent equal to the number in the box from which the cube was taken."

The initiative is not "taken from the player's pool and given to opponent." The Initiative pawn moves along a shared track — there is no per-player pool being transferred.

NUMBERS_CHECKED: 2 (initiative track span "20", order cube count "ten"), SUPPORTED: 2 (both confirmed in source); all procedural claims checked: 5, MAJOR errors found: 4
